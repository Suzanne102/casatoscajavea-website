# Casa Tosca Javea Website

This repository contains the website for Casa Tosca Javea, a vacation rental property in Jávea, Spain.

## 🌐 Live Website
https://www.casatoscajavea.com

## 📁 Project Structure
- `index.html` – homepage
- `css/` – styles
- `js/` – scripts
- `images/` – images
- `pages/` – internal pages
- `templates/` – shared components (nav, footer, etc.)
- `form-handlers/` – backend form processing (PHP)

## 🚀 Deployment
The website is deployed via GitHub to Hostinger using Git integration.

## 🛠️ Development
- Static HTML, CSS, JavaScript
- No build tools required
- Can be run locally using VS Code Live Server

## 🔒 Notes
Sensitive files and uploads are excluded via `.gitignore`.
